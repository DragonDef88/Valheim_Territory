Суть:

Сначала создаётся полное представление состояния мира в памяти, и только затем оно сериализуется.

Это даст нам большие преимущества:

можно экспортировать состояние без файлов;
можно позже отправлять Snapshot по сети;
можно делать предпросмотр изменений;
можно писать тесты без файловой системы
