# Clan Territory

A modular territory framework for Valheim.

## Status

🚧 Under Development
##Clan Territory Manifesto v1.0

Clan Territory is not built to be the biggest Valheim mod.

It is built to be the most maintainable one.

We value architecture over shortcuts.

We value correctness over speed.

We believe that every system should have one responsibility.

We believe the game world is the source of truth.

We believe that a Territory exists because a Ward exists.

Every sprint should leave the project in a better state than before.

We build for years, not for today.
